# ECCP 2026 — Org Admin User Guide

## Command Center Overview

### Daily Tasks
1. Check dashboard stats (scholars, attendance, profiles)
2. Review **Platform History** for security and compliance
3. Send **Reminders** to inactive scholars
4. Create or edit **global sessions** for the day

### User Management
| Action | How |
|--------|-----|
| Add scholar | Users tab → Add New Scholar form |
| Remove scholar | Users tab → Deactivate |
| Assign mentor | Edit scholar or set during creation |
| Reset password | Click Reset → copy credentials from modal |
| Send credentials | Click Credentials → copy email body |

### Session Management
- **Create** global sessions with topic, description, links
- **Edit** any existing session (topic, description, dates, links)
- **Delete** sessions that are no longer needed

### Program Timeline
- **Timeline tab** → Add/Edit phases visible to all users
- Keep aligned with ECCP 2026 bootcamp schedule

### Communication
- Broadcast to all scholars or all mentors
- Configure **Email Settings** (SMTP) for automatic reminders

### Reports & Exports
- Export scholar profiles (CSV)
- Export rankings (CSV)
- View mentor weekly reports

### Security
- Review login attempts in Platform History
- All admin actions logged in audit trail
- Set strong JWT_SECRET when deploying online
