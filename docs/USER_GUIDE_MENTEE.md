# ECCP 2026 — Scholar (Mentee) User Guide

## Step-by-Step Platform Usage

### Step 1: Login
1. Open the ECCP platform URL
2. Click **Scholar** on the home page
3. Enter your **PF Number** (e.g. 56647)
4. Password: `Cohort@2026` (change immediately in Profile)

### Step 2: Complete Profile (Required)
1. Go to **My Profile**
2. Fill: phone, school, subjects, career interests, goals
3. Complete **Application Status Tracker** — list universities you plan to apply to
4. Check AI ethics acknowledgment and mentor contact acknowledgment
5. Click **Save Profile**

### Step 3: Daily Routine
1. Open **Dashboard** — read today's topic and daily motivation quote
2. Go to **Sessions** — click **Join Session** if a link is provided
3. Wait for mentor to confirm your attendance
4. Submit **Daily Reflection** (rating + feelings + questions — all required)
5. Take the **Timed Quiz** before it closes (missing it affects your ranking)

### Step 4: Handle Absences (If Applicable)
If you miss a session:
1. You will see the **Responsibility Corner** gate when trying to access certain features
2. Provide a minimum 100-character professional explanation for your absence
3. Submit the excuse to regain access to university portals, SAT results, and dashboards

### Step 5: Use Academic Preparation Tools
- **SAT Preparation Simulator**: Practice SAT-style questions with instant feedback
- **STAR Activity Optimizer**: Craft college-application activity descriptions within character limits
- **Global Opportunities Tracker**: Browse scholarships, fellowships, and academic programs worldwide

### Step 6: Use Resources
- Browse **Resource Library** by category: SAT Prep, Essay Writing, etc.
- Open links, PDFs, and YouTube videos shared by mentors

### Step 7: Stay Connected
- **Messages** — filter Org Admin vs Your Mentor messages; reply when needed
- **Know Your Mentor** — view bio and social links
- **SAT Prep** — complete practice exams and submit scores

### Step 8: Track Applications
- Update profile when you start applying to universities
- If admitted, set status to "Admitted" and enter university name

### Special Situations
- **Administrative Hold**: If you see a security gate, contact your mentor to schedule an advising review session to restore access
- **Password Reset**: Use "Forgot password" — link sent to your registered email
